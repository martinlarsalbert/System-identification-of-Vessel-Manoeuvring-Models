3. Inverse dynamics¶

Each manoeuvring model has some hydrodynamic functions \(X_D(u,v,r,\delta,thrust)\), \(Y_D(u,v,r,\delta,thrust)\), \(N_D(u,v,r,\delta,thrust)\) that are defined as polynomials.

The hydrodynamic derivatives in these polynomials can be identified with force regression of measured forces and moments. The measured forces and moments are usually taken from captive model tests, PMM tests or Virtual Captive Tests (VCT) being the virtual version of captive/PMM tests calculated with Computational Fluid Dynamics (CFD).

When the ship or ship model is free in all degrees of freedome, as in the present model tests, only motions can be observed however. The forces and moments that generated the motions needs to be estimated by solving the inverse dynamics problem and doing a motion regression.

The inverse dynamics is solved by restructering the system equation (3.1) to get the hydrodynamics functions on the left hand side (3.2), (3.3), (3.4). If the mass and intertia of the ship as well as added masses: \(\dot{u}\), \(\dot{v}\), \(\dot{r}\) are known, the forces can be calculated using these equations.

(3.1)¶\[\begin{split}\displaystyle \left[\begin{matrix}- X_{\dot{u}} + m & 0 & 0\\0 & - Y_{\dot{v}} + m & - Y_{\dot{r}} + m x_{G}\\0 & - N_{\dot{v}} + m x_{G} & I_{z} - N_{\dot{r}}\end{matrix}\right] \left[\begin{matrix}\dot{u}\\\dot{v}\\\dot{r}\end{matrix}\right] = \left[\begin{matrix}m r^{2} x_{G} + m r v + \operatorname{X_{D}}{\left(u,v,r,\delta,thrust \right)}\\- m r u + \operatorname{Y_{D}}{\left(u,v,r,\delta,thrust \right)}\\- m r u x_{G} + \operatorname{N_{D}}{\left(u,v,r,\delta,thrust \right)}\end{matrix}\right]\end{split}\]
(3.2)¶\[\displaystyle \operatorname{X_{D}}{\left(u,v,r,\delta,thrust \right)} = - X_{\dot{u}} \dot{u} + \dot{u} m - m r^{2} x_{G} - m r v\]
(3.3)¶\[\displaystyle \operatorname{Y_{D}}{\left(u,v,r,\delta,thrust \right)} = - Y_{\dot{r}} \dot{r} - Y_{\dot{v}} \dot{v} + \dot{r} m x_{G} + \dot{v} m + m r u\]
(3.4)¶\[\displaystyle \operatorname{N_{D}}{\left(u,v,r,\delta,thrust \right)} = I_{z} \dot{r} - N_{\dot{r}} \dot{r} - N_{\dot{v}} \dot{v} + \dot{v} m x_{G} + m r u x_{G}\]

4. Regression¶

Finding the the hydrodynamic derivatives can be defined as a linear regression problem:

(4.1)¶\[y = X\beta + \epsilon\]

A model for the hydrodynamic forces first needs to be assumed for instance as the polynomials in: (4.2), (4.3) and (4.4).

(4.2)¶\[\displaystyle \operatorname{X_{D}}{\left(u,v,r,\delta,thrust \right)} = X_{deltadelta} \delta^{2} + X_{rr} r^{2} + X_{thrust} thrust + X_{u} u + X_{vr} r v\]
(4.3)¶\[\displaystyle \operatorname{Y_{D}}{\left(u,v,r,\delta,thrust \right)} = Y_{delta} \delta + Y_{r} r + Y_{thrustdelta} \delta thrust + Y_{ur} r u + Y_{u} u + Y_{v} v\]
(4.4)¶\[\displaystyle \operatorname{N_{D}}{\left(u,v,r,\delta,thrust \right)} = N_{delta} \delta + N_{r} r + N_{thrustdelta} \delta thrust + N_{ur} r u + N_{u} u + N_{v} v\]

The label vector \(y\) and feature matrix \(X\) in the regression problem in (4.1) can now be inserted. As an example the regression in surge degree of freedome the labels can be calculated using the inverse dynamics force:

(4.5)¶\[\displaystyle y = - X_{\dot{u}} \dot{u} + \dot{u} m - m r^{2} x_{G} - m r v\]

And the feature matrix \(X\) is expressed as:

(4.6)¶\[\displaystyle X = \left[\begin{matrix}thrust & u & \delta^{2} & r^{2} & r v\end{matrix}\right]\]

And the regressed hydrodynamic derivatives are stored in the \(\beta\) vector:

(4.7)¶\[\begin{split}\displaystyle \beta = \left[\begin{matrix}X_{thrust}\\X_{u}\\X_{deltadelta}\\X_{rr}\\X_{vr}\end{matrix}\right]\end{split}\]

Solving the regressions with OLS in each degree of freedome gives a perfect match of hydrodynamic derivatives compared to the real model used when generating the simulated data as seen in fig. Parameter identification using linear regression with forces calculated with inverse dynamics with a model that is perfectly correct on data that is also perfect will obviously give a perfect identification of the system.
